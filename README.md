# SimpleToDo

SimpleToDo 

Use the [dotnet 5.0](https://dotnet.microsoft.com/en-us/download/dotnet/5.0)

## Usage

```c#
// install dependencies
dotnet restore

// build
dotnet build

// run
dotnet watch run
```
